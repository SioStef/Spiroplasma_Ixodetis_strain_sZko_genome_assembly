# Spiroplasma_Ixodetis_strain_sZko_genome
This repository contains the genome assembly of Spiroplasma ixodetis strain sZko, associated with the drosophilid Zaprionus kolodkinae, along with corresponding annotation data.

The genome assembly was generated from publicly available sequencing datasets deposited in the NCBI Sequence Read Archive (SRA) under BioProject PRJNA675888 (accessions SRR13070731 and SRR13070681) (see https://doi.org/10.7554/eLife.66405)
